# Administración de claves GPG: información general

GPG (GnuPG) cifra los mensajes utilizando pares de claves individuales asimétricos. Para implementar el cifrado GPG con [!DNL Campaign], un usuario administrador debe instalar o generar claves GPG en una instancia de marketing directamente desde el Panel de control.
Obtenga información sobre cómo generar e instalar un par de claves pública y privada en una instancia [!DNL Campaign] específica para el cifrado de datos salientes. Aprenda cómo importar e instalar una clave pública en una instancia de [!DNL Campaign] para el descifrado de datos entrantes.

* [Uso de una clave GPG para cifrar datos](./use-a-gpg-key-to-encrypt-data.md)
